dotfiles
========

Personal dotfiles for both Ubuntu & OSX.

Currently designed to work for:

  * **Kubuntu 13.04**
  * **Mac OSX 10.8 (Mountain Lion)**

Snapshots of older distributions may be found as tags of the current repository.


Ubuntu install
--------------

Run:

    $ ./ubuntu-install.sh


OSX install
-----------

Run:

    $ ./osx-install.sh
